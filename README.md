AI Tutor — умный помощник для обучения и изучения.

## Возможности

- Современный интерфейс
- Поддержка различных тем
- Адаптивный дизайн
- Быстрая работа

## Установка

1. Клонируй репозиторий:
```bash
git clone https://github.com/botir07/aitutor.git
```

2. Открой файл `index.html` в браузере

## Использование

Просто открой файл `index.html` в любом современном браузере.

## Структура проекта

```
aitutor/
├── index.html       # Главный файл приложения
└── README.md        # Документация
```

## Разработка

Для разработки можно использовать любой текстовый редактор. Приложение написано на чистом HTML/CSS/JavaScript.
